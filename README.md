# Ruum Ruum Conductor - Next.js

Aplicación refactorizada desde un HTML estático a un proyecto Next.js con React, TypeScript y Tailwind CSS.

## Ejecutar localmente

```bash
npm install
npm run dev
```

Luego abre `http://localhost:3000`.

## Qué se migró

- Navegación inferior entre Panel, Viajes, Ganancias y Perfil usando estado React.
- Toggle de disponibilidad controlado por estado.
- Pestañas de viajes solicitados y aceptados.
- Acciones de aceptar/rechazar viaje.
- Modal de carga de evidencia.
- Estilos migrados desde Tailwind CDN a Tailwind local del proyecto.
